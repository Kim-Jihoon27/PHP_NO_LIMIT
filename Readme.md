# PHP NO LIMIT

## Overview
**PHP NO LIMIT** is an open-source project where anyone can create PHP problems and share their solutions. It's a place to learn and get better at solving problems with PHP through hands-on practice.

This project was created by **Kim Jihoon** to help people learn PHP by doing real challenges.

## Purpose
- Practice solving problems in PHP  
- Let contributors create their own challenges  
- Share and improve solutions together  
- Help beginners build their PHP skills with practical examples  

## Features
- **Open to Everyone**: Add your own problems or solutions anytime  
- **Clear Structure**: Each problem is linked directly to its solution  
- **Learn by Doing**: Focus on PHP basics and real problem-solving
